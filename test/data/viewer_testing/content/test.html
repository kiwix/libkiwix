Testing Guidelines

  • Test in different browsers (Firefox, Chromium, desktop, mobile).
  • Test with external link blocking disabled/enabled.
  • Links activation can be performed by
    • clicking
    • CTRL clicking
    • SHIFT clicking
    • middle clicking
    • selecting "Open Link in New Tab" from the context menu
    • selecting "Copy Link" from the context menu

Testing Status

In the tables below the Base case column represents the simplest setup of the test page (no external link blocking, no wrapping iframe, etc). Each subsequent column contains a list of (additional) problems observed for the respective feature enabled (possibly in combination with one or more of the features from the previous columns).

Issues referenced in the header of the column apply to the entire column or most of its rows.
